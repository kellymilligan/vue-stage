# vue-stage

A basic exmaple app showcasing vue-stage in action. Includes animation examples using JS and CSS, and includes an example with a highly nested component. 

---

## Installation

from the vue-stage base directory:

### 1.
```sh
$ cd examples
```

### 2. 
```sh
$ npm install
or
$ yarn add
``` 

### 3. 
```sh
$ npm run serve
or 
$ yarn run serve
```
